## ⚡ Overview
**EmirNetwork**, terminal sınırlarını zorlayan, Windows tabanlı bir ağ analiz ve istihbarat toolkit'idir. **True RGB Rainbow Engine** entegrasyonu ile sadece bir tool değil, aynı zamanda görsel bir deneyim sunar.

## 🚀 Key Features
* **Elite UI:** 24-bit True RGB dinamik renk geçişleri.
* **Intelligence:** Multithreaded port tarama ve gelişmiş servis tanıma.
* **OSINT:** Geo-IP, Whois ve DNS enumerasyon modülleri.
* **System Tracking:** Gerçek zamanlı CPU/RAM ve Network metrikleri.
