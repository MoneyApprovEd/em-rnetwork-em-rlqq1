## 🚀 Overview | Genel Bakış

**EmirNetwork**, Windows UTF-8 platformları için özel tasarlanmış, **True RGB Rainbow Engine** ile terminal görselliğini zirveye taşıyan profesyonel bir OSINT ve Ağ Analiz Platformudur.

## 🛠️ Modüller

- **RGB Rainbow Engine:** 24-bit dinamik renk geçişleri.
- **Port Scanner:** Çok izlekli (multithreaded) hızlı tarama.
- **Geo-IP & Whois:** Detaylı adres ve sahiplik sorgulama.
- **System Metrics:** Gerçek zamanlı işlemci ve bellek takibi.
